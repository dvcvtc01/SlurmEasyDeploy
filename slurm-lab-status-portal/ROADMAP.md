# Roadmap (Alpha)

## Near term
- Add optional API auth (basic token mode).
- Add unit tests for command parsers and error handling.
- Add CI checks (lint + syntax + smoke tests).
- Add packaging path for Debian-style service install.

## Mid term
- Add safe job-template submission UI.
- Add Slurm accounting integration (`slurmdbd` + DB).
- Add role-based access control and audit logging.
- Add cluster profile support for multi-cluster dashboards.

## Long term
- Add plugin model for scheduler backends.
- Add historical trend charts for jobs and host health.
- Add HA deployment guide.
